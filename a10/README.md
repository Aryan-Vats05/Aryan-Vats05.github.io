### Data modifications
Three sadness % values were changed from the original CDC data:

- 2010: 28.5 changed to 32.1
- 2018: 36.7 changed to 34.2
- 2022: 44.2 changed to 40.8

Goal was to make the overall trend look smaller to support the skeptical narrative about screen time.
